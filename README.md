# AI Response Evaluation & Annotation System

## Overview
This project provides a structured framework for evaluating and annotating AI-generated responses. It enables systematic labeling, quality scoring, inter-annotator agreement measurement, and confusion matrix analysis.

## Features
- Interactive Streamlit annotation interface
- Structured evaluation (Accuracy, Relevance, Completeness)
- Automated quality scoring
- Confusion matrix visualization
- Inter-annotator agreement calculation

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
- Matplotlib / Seaborn
